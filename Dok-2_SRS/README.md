# Software Requirement Specification (SRS)

## 1. Deskripsi Sistem
EMart merupakan aplikasi e-commerce berbasis web yang digunakan 
untuk melakukan pembelian produk secara online.

## 2. Aktor Sistem
1. Admin
2. Customer

## 3. Kebutuhan Fungsional

### Admin
- Admin dapat login
- Admin dapat menambah produk
- Admin dapat mengedit produk
- Admin dapat menghapus produk
- Admin dapat melihat transaksi

### Customer
- User dapat register
- User dapat login
- User dapat melihat produk
- User dapat add to cart
- User dapat checkout

## 4. Kebutuhan Non Fungsional
- Sistem berbasis web
- Sistem berjalan pada browser
- Sistem membutuhkan koneksi internet
- Sistem menggunakan database

## 5. Batasan Sistem
- Tidak ada pembayaran otomatis
- Tidak ada chat
- Tidak ada fitur rating
