# User Documentation

## 1. Pendahuluan
Dokumen ini berisi panduan penggunaan sistem EMart bagi pengguna.

## 2. Cara Login
1. Buka website EMart
2. Klik login
3. Masukkan username dan password
4. Klik login

## 3. Cara Register
1. Klik register
2. Isi data user
3. Klik submit
4. Akun berhasil dibuat

## 4. Cara Melihat Produk
1. Login ke sistem
2. Masuk halaman produk
3. Pilih produk

## 5. Cara Add to Cart
1. Pilih produk
2. Klik add to cart
3. Produk masuk keranjang

## 6. Cara Checkout
1. Masuk ke cart
2. Klik checkout
3. Transaksi berhasil

## 7. Logout
Klik tombol logout untuk keluar dari sistem
# Interface Login EMart

Halaman login digunakan untuk mengakses sistem EMart. 
User harus memasukkan username dan password untuk masuk ke sistem.

Komponen pada halaman login:
1. Field Username
2. Field Password
3. Tombol Login

Tujuan Interface:
Memudahkan user melakukan autentikasi sebelum masuk ke sistem.
