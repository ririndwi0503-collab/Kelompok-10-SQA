*Kelompok - 10 SQA*
_Dokumentasi Software Quality EMart_

Repository ini digunakan untuk mendokumentasikan proses Software Quality Testing pada aplikasi EMart

_Metode Pengujian:_
1. White Box Testing
2. Black Box Testing
3. Grey Box Testing

_Objek Pengujian:_
- login.html
- register.html
- home.html
- script.js
- style.css

_Tujuan Pengujian_ 
- Memastikan sistem berjalan sesaui kebutuhan pengguna.
- Menemukan bug atau kesalahan pada sistem
- Meningkatkan kualitas perangkat lunak

_Fungsi Github_ 
Github digunakan sebagai medida komunikasi antara programmer dan software quality tester, penyimpanan dokumen pengujian, pencatatan bug, serta dokumentasi hasil pengujian sebelum dibuat final report.
