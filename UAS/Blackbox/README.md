# Black Box Testing

Black Box Testing adalah metode pengujian yang dilakukan dengan menguji fungsi sistem berdasarkan input dan output tanpa melihat struktur kode program.

## Metode Pengujian

1. Boundary Value Analysis
2. Equivalence Partitioning
3. Decision Table Testing
4. Cause Effect Relationship Testing
5. Sample Testing

## Tujuan

- Memastikan fungsi sistem berjalan sesuai kebutuhan.
- Memvalidasi input dan output sistem.
- Menemukan kesalahan fungsional pada aplikasi.
- Memastikan kebutuhan pengguna terpenuhi.
