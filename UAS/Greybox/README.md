# Grey Box Testing

Grey Box Testing adalah metode pengujian yang menggabungkan pendekatan White Box Testing dan Black Box Testing. Penguji memahami sebagian struktur internal sistem serta menguji fungsi eksternal aplikasi.

## Metode Pengujian

1. Matrix Testing
2. Orthogonal Array Testing
3. Pattern Testing
4. Regression Testing

## Tujuan

- Menguji fungsi sistem dengan pemahaman sebagian struktur internal.
- Meningkatkan efektivitas pengujian.
- Menemukan kesalahan integrasi sistem.
- Memastikan perubahan sistem tidak menimbulkan bug baru.
