# Desk Checking

## Definisi

Desk Checking merupakan teknik White Box Testing yang dilakukan dengan memeriksa logika program secara manual tanpa menjalankan aplikasi.

## Tujuan

- Memeriksa logika program.
- Menemukan kesalahan algoritma.
- Memastikan proses berjalan sesuai kebutuhan.

## Objek Pengujian

Fitur Login pada sistem EMart.

## Penjelasan Pengujian

Program diperiksa langkah demi langkah menggunakan data simulasi untuk memastikan proses validasi login berjalan dengan benar.

## Simulasi Data

| Username | Password | Hasil |
|-----------|-----------|---------|
| admin | 12345 | Login Berhasil |
| admin | salah | Login Gagal |

## Hasil Pengujian

Logika program menghasilkan output sesuai dengan data yang diberikan.

## Kesimpulan

Desk Checking menunjukkan bahwa logika login telah berjalan sesuai kebutuhan sistem.
