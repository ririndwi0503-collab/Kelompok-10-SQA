# White Box Testing

White Box Testing adalah metode pengujian perangkat lunak yang berfokus pada struktur internal, logika program, dan alur kode sumber aplikasi.

## Metode Pengujian

1. Desk Checking
2. Data Flow Testing
3. Control Flow Testing
4. Code Walkthrough
5. Formal Inspection

## Tujuan

- Memeriksa logika program.
- Memastikan alur proses berjalan dengan benar.
- Mengidentifikasi kesalahan pada struktur kode.
- Meningkatkan kualitas perangkat lunak.
