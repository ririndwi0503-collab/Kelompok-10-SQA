# Data Flow Testing

## Definisi

Data Flow Testing merupakan metode White Box Testing yang digunakan untuk memeriksa aliran data dari input, proses, hingga menghasilkan output pada sistem.

## Tujuan

- Memastikan data diproses dengan benar.
- Mengidentifikasi kesalahan penggunaan variabel.
- Memastikan output sesuai dengan data input.

## Objek Pengujian

Fitur Login pada sistem EMart.

## Penjelasan Pengujian

Pengujian dilakukan terhadap aliran data username dan password yang dimasukkan pengguna hingga menghasilkan status login.

## Tabel Aliran Data

| Tahap | Data |
|---------|---------|
| Input | Username dan Password |
| Proses | Validasi Data |
| Output | Login Berhasil / Gagal |

## Hasil Pengujian

Data berhasil mengalir dari proses input hingga output tanpa ditemukan kesalahan penggunaan variabel.

## Kesimpulan

Aliran data pada fitur Login berjalan sesuai dengan kebutuhan sistem.
